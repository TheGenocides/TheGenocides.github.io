As a programmer, I really enjoy making project and as a result I have made tons of projects! Most of my project are written using Python since Python is my first ever programming language. Here are some projects that I have created!

PyTweet


An object oriented programming library for the twitter apis written in python.


26 Stars | 13 Forks

PyTweet is my favorite project that I have worked in with few of my friends. It is an open source library for making request with twitter api(s). Supporting Account Activity API, Api version 1.1 & version 2 which is the newest version. Currently the project has been archived. Feel free to fork and make some changes for your personal needs!

Holybooks


A synchronous object oriented library for the Quran & Bible apis.


5 Stars | 3 Forks

Holybooks is a library or an api wrapper for making requests to Bible and Quran api(s). I think this is one of my favorite project because personally I really enjoy studying religion. This library is still being maintain and is being rewrite with new features and even new api support!

PyCommands


A synchronous package for making simple and fun commands in python.


3 Stars | 2 Forks

Pycommands is a very unique and synchronous library that I made using Python. It is a fun library mocking CLI commands but with custom builtint listener in Python and a very cool terminal styled with colors!
